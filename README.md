# Generative-Agentic-AI-Project
This project presents a generative agentic AI prototype designed to address operational inefficiencies within a retail organization, SmartShoe using Stack AI. The system leverages multi-agent coordination to streamline workflows, improve data exchange, and support automated decision-making.
